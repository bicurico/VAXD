# VAXD

<p align="left">
  <img src="screenshots/VAXD%20-%20Small.png" alt="VAXD Logo" width="180">
</p>

**VAXD** is a lightweight Windows disassembler, PE inspection tool, and patch-assistance environment for native Windows executable files.

It is designed for fast analysis of Windows PE EXE and DLL files, with a practical workflow focused on disassembly, navigation, strings, jumps, calls, patch planning, pseudo-code generation, and binary patching assistance.

VAXD focuses on speed, simplicity, and usability. It is intended for users who want a quick and focused tool for inspecting native Windows executables without the complexity and overhead of larger reverse-engineering suites.

## What VAXD is for

VAXD can be useful for:

- inspecting Windows EXE and DLL files
- checking what a file actually is, even when the extension is missing or misleading
- reviewing PE headers, sections, imports, exports, strings, and resources
- navigating disassembled code quickly
- finding strings and code references
- reviewing jumps, calls, and code flow
- planning and managing binary patches
- fixing or modifying old Windows applications
- security research, malware triage, and software auditing
- reverse-engineering education and experimentation

VAXD is not intended to replace large professional reverse-engineering platforms. Its goal is to provide a faster, simpler, and more direct workflow for many common executable-inspection and patch-assistance tasks.
